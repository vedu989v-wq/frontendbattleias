# Ravindra's IAS Institute

A professional website for Ravindra's UPSC Coaching Institute in Bhopal.

## Features

- Responsive design with sticky scroll sections
- Hero carousel with automatic transitions
- Topper showcase with navigation
- FAQ section with toggle functionality
- Comprehensive course information
- Modern UI with smooth animations

## Setup & Testing Locally

**DO NOT** open `index.html` directly by double-clicking. Always use a local server:

### Option 1: Python
```bash
python -m http.server 8000
```
Then open: `http://localhost:8000`

### Option 2: Node.js (recommended)
```bash
npx http-server
```

## Deployment on Vercel

1. Push code to GitHub
2. Connect repository to Vercel
3. Vercel will automatically detect and deploy as static site
4. Your site will be live at `your-vercel-domain.vercel.app`

## Files

- `index.html` - Home page with hero, stats, toppers, features, and FAQ
- `about.html` - About the institute
- `courses.html` - Available courses
- `study_resources.html` - Study materials and resources
- `vercel.json` - Vercel deployment configuration
- `package.json` - Project metadata

## Browser Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
